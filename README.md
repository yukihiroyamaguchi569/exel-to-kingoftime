# Excel → KingOfTime 変換ツール

Excelの勤務表をKingOfTime用CSVに変換するWebアプリケーションです。

## 概要

勤務表のExcelファイルをアップロードし、視覚的なパイプラインUIで変換ステップを組み合わせることで、KingOfTimeにインポート可能なCSVを生成します。

## 機能

- **Excelアップロード** - ドラッグ&ドロップまたはクリックでアップロード（.xlsx / .xls / .xlsm / .ods 対応）
- **シート選択** - 複数シートのExcelに対応
- **自動バリデーション** - 列数、日付ヘッダー形式、社員番号、空白セルを自動チェック
- **変換パイプライン** - 8種類のステップをドラッグ&ドロップで並び替えて適用
- **プレビュー** - 変換前・変換後・参照ファイルをタブで確認（先頭50行）
- **CSVダウンロード** - ヘッダー行なし・UTF-8でエクスポート

### パイプラインステップ

| ステップ | 説明 |
|---|---|
| アンピボット (Melt) | 横持ちデータを縦持ちに変換 |
| 左結合 (Left Join) | 参照ファイルと結合（シフト→パターンコード変換） |
| 行削除 | 指定条件または行番号範囲で行を削除 |
| 列削除 | 指定列を削除 |
| 行フィルター | 条件に一致する行のみ残す |
| 列並び替え | 列の順序を変更 |
| ゼロ埋め | 数値を指定桁数でゼロ埋め |
| 年月付与 | 列の値の先頭にYYYYMMを付与 |

### デフォルトパイプライン

ファイル読み込み時に以下のパイプラインが自動セットされます。

```
1. アンピボット         (縦持ち変換)
2. 左結合              (シフトコード → パターンコード)
3. 列削除              (「シフト」列を削除)
4. 行削除              (パターンコードがnullの行を削除)
5. 列並び替え          (日付・社員番号・パターンコード順)
6. ゼロ埋め            (日付を2桁)
7. ゼロ埋め            (社員番号を3桁)
8. 年月付与            (日付にYYYYMMを付与)
```

## セットアップ・起動

### 必要環境

- Python 3.9 以上

### 起動

```bash
./run.sh
```

スクリプトが自動的に以下を実行します。
1. Python バージョン確認
2. 仮想環境 (`.venv`) の作成・有効化
3. 依存パッケージのインストール
4. Uvicornサーバー起動

ブラウザで http://localhost:8000 を開いてください。

### 手動起動

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r backend/requirements.txt
cd backend
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### CLI（スタンドアロン変換）

```bash
python convert.py <input.xlsx> [output.csv]
```

## 使い方

1. **勤務表をアップロード** - Excelファイルをドラッグ&ドロップ
2. **シートを選択** - 複数シートがある場合は対象シートを選択
3. **参照ファイルをアップロード** - シフト名→パターンコードのマッピングファイルをアップロード
4. **バリデーション確認** - 自動チェック結果を確認
5. **パイプラインを調整** - 必要に応じてステップを追加・削除・並び替え
6. **プレビューで確認** - 変換結果を確認
7. **CSVダウンロード** - ダウンロードボタンでCSVを出力
